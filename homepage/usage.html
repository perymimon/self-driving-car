Usage Instructions

Setting up and running the Self-Driving Car Project is straightforward, with no dependencies required. Just follow the steps below to get started.

Setup Instructions

To get started with the project, simply:

  1. Clone the Repository: Clone the project repository from GitHub: GitHub Repository.
  2. Copy the Main Folder: After cloning, copy the main folder to your desired location.
  3. Open an HTML File: Navigate to the folder and click on any of the HTML files to open them in your web browser.

Running Simulations

The project includes several simulations that you can run by opening the respective HTML files:

  • 2D Car Simulation: Open the 2D simulation by clicking on 2D Simulation.
  • 3D Car Simulation: Open the 3D simulation by clicking on 3D Simulation.

Controlling the Car

You can control the car using various methods depending on the simulation you're running:

  • Keyboard Controls: Use the arrow keys on your keyboard to control the car's movement in the 3D simulation.
  • Camera Controls: If you're using the Race 3D Using Computer Camera module, position a blue object in front of your camera to control the car.
  • Phone Controls: For the Race 3D Using Sensors on Phone, tilt and rotate your phone to control the car's movement (feature under development).

For detailed source code and further customization, visit the GitHub Repository.
